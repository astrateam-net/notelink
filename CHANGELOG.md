# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-12-19

### Added
- Russian language support (i18n).
- Language override setting in plugin settings (Auto / English / Русский).
- All UI strings are now translatable.

### Fixed
- TypeScript type issues with moment import.
- ESLint warnings in ViewModes interface.

## [1.0.0] - 2025-12-19

### Added
- Initial release of NoteLink plugin.
- Rebranding from Share Note to NoteLink.
- Updated API endpoints to point to `notelink-api`.
- Support for detailed theme upload.
- Encrypted note sharing logic.
